## [1.0.0] — 2025

### Added
- Initial release of Signature CSS Framework.
- Zero-dependency vanilla CSS architecture with CSS Custom Properties.
- Native Bi-Directional (RTL/LTR) support via CSS logical properties.
- Glassmorphism dark mode and ceramic light mode themes.
- Core components: buttons, forms, tables, cards, badges, modals,
  spinners, toasts, and sidebar navigation.
- Full showcase demo (`index.html`) with live RTL/LTR toggle.
